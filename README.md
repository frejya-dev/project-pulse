# Project Pulse

Project Pulse is a fictional infrastructure operations dashboard built with React.

I wanted to build something that felt closer to the kinds of internal tools software engineering and infrastructure teams use instead of another weather app or to-do list. The dashboard brings together project tracking, operational status, search, and reporting in a clean interface inspired by transportation and engineering organizations.

## Features

- Infrastructure operations dashboard
- Project status overview
- Search projects by name
- Filter projects by status
- Monthly delivery trend visualization
- Responsive layout
- Component-based React architecture

## Why I Built This

I'm expanding my frontend portfolio with projects that reflect the kinds of applications I'd like to build professionally.

This project gave me an opportunity to practice breaking a larger interface into reusable React components, managing application state with hooks, organizing project data into separate modules, and building a dashboard that feels like something a real operations team could use.

## What I Practiced

- Building reusable React components
- Managing state with `useState`
- Passing data through props
- Conditional rendering
- Filtering and mapping arrays
- Organizing application data into separate modules
- Creating responsive layouts with CSS
- Visualizing data with Recharts
- Deploying a React application with AWS Amplify
