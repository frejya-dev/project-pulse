# Project Pulse

Project Pulse is a fictional infrastructure operations dashboard built with React.

The application is designed to explore modern frontend development by simulating the type of internal software an engineering or transportation organization might use to monitor projects, operational metrics, and infrastructure status.

All projects, metrics, alerts, and data shown in the application are fictional.

---

## Why I Built This

I wanted to strengthen my React skills by building a realistic application instead of following a tutorial.

After learning more about the types of software used in engineering and infrastructure organizations, I decided to create a fictional operations dashboard that emphasizes clean UI design, reusable components, and modern frontend architecture.

The goal is to practice building software that feels like a real internal business application while continuing to expand my frontend development skills.

---
